# 树状数组 (Fenwick Tree / Binary Indexed Tree) 思考文档

## 一、概述

**树状数组**（Binary Indexed Tree，BIT），也称 **Fenwick 树**，由 Peter M. Fenwick 于 1994 年提出。它是一种用于高效维护**前缀和**（Prefix Sum）的数据结构，支持：

- **单点更新**（给某位置加一个值）：O(log n)
- **前缀和查询**（求前 k 个元素之和）：O(log n)

在最坏情况下，朴素数组的更新或查询至少有一者为 O(n)，而树状数组将两者都优化到了对数级别。

---

## 二、核心思想

### 2.1 lowbit 运算

树状数组最重要的概念是 `lowbit(x)`，定义为 **x 的二进制表示中最低位的 1 所对应的数值**。

```text
lowbit(x) = x & (-x)
```

示例：

| x   | 二进制    | lowbit(x) |
|-----|-----------|-----------|
| 1   | 0001      | 1         |
| 2   | 0010      | 2         |
| 3   | 0011      | 1         |
| 4   | 0100      | 4         |
| 6   | 0110      | 2         |
| 8   | 1000      | 8         |

### 2.2 存储结构

设原数组为 `a[1..n]`（注意：树状数组通常使用 **1-indexed**），树状数组 `bit[1..n]` 的定义为：

```
bit[i] = 从位置 i 开始，向前数 lowbit(i) 个元素之和
       = a[i - lowbit(i) + 1] + ... + a[i]
```

也就是说，`bit[i]` 负责维护区间 `(i - lowbit(i), i]` 的和。

**关键观察**：`bit[i]` 的覆盖范围恰好是 `lowbit(i)`。

### 2.3 树状结构图

```
        bit[8] ──────────────────── a[1..8]
      /        \
  bit[4]        bit[6]         bit[7]
  a[1..4]      a[5..6]        a[7..7]
 /    \        /    \          |
bit[2] bit[3] bit[5]          a[8]
a[1..2] a[3..3] a[5..5]
/ \
a[1] a[2]
```

每个节点 `bit[i]` 的**父节点**是 `i + lowbit(i)`。

## 三、基本操作

### 3.1 初始化

将 `bit` 数组初始化为全 0，然后对每个位置 i 执行一次 `add(i, a[i])`，总复杂度 O(n log n)。

优化初始化（O(n)）：

```cpp
for (int i = 1; i <= n; i++) {
    bit[i] += a[i];
    int j = i + (i & -i);
    if (j <= n) bit[j] += bit[i];
}
```

### 3.2 单点更新：add(pos, delta)

给 `a[pos]` 加上 `delta`，所有包含 `pos` 的 `bit[i]` 都要更新。

```cpp
void add(int pos, int delta) {
    while (pos <= n) {
        bit[pos] += delta;
        pos += pos & -pos;      // 走到父节点
    }
}
```

原理：从 `pos` 开始，不断加上自己的 `lowbit` 跳到父节点，直到越界。

### 3.3 前缀和查询：sum(pos)

求 `a[1] + a[2] + ... + a[pos]`。

```cpp
int sum(int pos) {
    int res = 0;
    while (pos > 0) {
        res += bit[pos];
        pos -= pos & -pos;      // 去掉最低位的 1
    }
    return res;
}
```

原理：从 `pos` 开始，不断减去自己的 `lowbit`，累加沿途的 `bit` 值。

### 3.4 区间和查询

求 `a[l] + ... + a[r]`：

```cpp
int range_sum(int l, int r) {
    return sum(r) - sum(l - 1);
}
```

### 3.5 查询 & 修改某点的值

```cpp
int get(int pos) {
    return sum(pos) - sum(pos - 1);
}

void set(int pos, int val) {
    int cur = get(pos);
    add(pos, val - cur);
}
```

## 四、进阶功能

### 4.1 区间更新 + 单点查询（差分数组）

利用差分思想：设差分数组 `d[i] = a[i] - a[i-1]`（设 `a[0] = 0`）。

- **区间更新**：对 `a[l..r]` 每个元素加 `v` → 在差分数组中 `d[l] += v`, `d[r+1] -= v`
- **单点查询**：`a[pos] = sum(d[1..pos])`

用树状数组维护差分数组即可。

```cpp
// 在 [l, r] 区间加 v
void range_add(int l, int r, int v) {
    add(l, v);
    add(r + 1, -v);
}

// 查询位置 pos 的值
int point_query(int pos) {
    return sum(pos);
}
```

### 4.2 区间更新 + 区间查询（二阶树状数组）

维护两个 BIT：

- 设原数组差分数组为 `d[i]`
- 前缀和 `sum(a[1..p])` = `(p+1) * sum(d[1..p]) - sum(i * d[i], 1..p)`

维护两个 BIT：
- `bit1` 维护 `d[i]`
- `bit2` 维护 `i * d[i]`

```cpp
int bit1[N], bit2[N];

void range_add(int l, int r, int v) {
    add(bit1, l, v);
    add(bit1, r + 1, -v);
    add(bit2, l, l * v);
    add(bit2, r + 1, -(r + 1) * v);
}

int prefix_sum(int p) {
    return (p + 1) * sum(bit1, p) - sum(bit2, p);
}

int range_sum(int l, int r) {
    return prefix_sum(r) - prefix_sum(l - 1);
}
```

### 4.3 求第 k 小的值（权值树状数组 + 二分/倍增）

如果树状数组维护的是**频次数组**（即值域 BIT），可以用倍增法找到**前缀和首次 >= k** 的位置：

```cpp
int kth(int k) {   // 找到前缀和 >= k 的最小下标
    int pos = 0;
    for (int d = highest_power_of_two; d > 0; d >>= 1) {
        int nxt = pos + d;
        if (nxt <= n && bit[nxt] < k) {
            k -= bit[nxt];
            pos = nxt;
        }
    }
    return pos + 1;
}
```

复杂度 O(log n)，比二分再查询 O(log² n) 更优。

### 4.4 多维树状数组

二维情况：维护一个二维 BIT，单点更新为 `(x, y)`，查询为 `(1..x, 1..y)` 的子矩阵和。

```cpp
void add(int x, int y, int delta) {
    for (int i = x; i <= n; i += i & -i)
        for (int j = y; j <= m; j += j & -j)
            bit[i][j] += delta;
}

int sum(int x, int y) {
    int res = 0;
    for (int i = x; i > 0; i -= i & -i)
        for (int j = y; j > 0; j -= j & -j)
            res += bit[i][j];
    return res;
}
```

复杂度 O(log n × log m)。可扩展到更高维，维度 d 时复杂度 O(log^d n)。

## 五、与线段树的对比

| 特性 | 树状数组 | 线段树 |
|------|----------|--------|
| 实现难度 | 非常简单（~10 行） | 中等（~50-80 行） |
| 时间复杂度 | O(log n) | O(log n) |
| 空间 | O(n) | O(4n) 或 O(2n) |
| 常数因子 | 极小 | 较大 |
| 区间查询 | 仅支持前缀和可逆运算（和、异或、乘积） | 支持任意结合律操作（max, min, gcd 等） |
| 区间更新 | 需差分技巧（加减） | 支持懒标记，任意区间修改 |
| 可扩展性 | 较差 | 很强（线段树合并、动态开点等） |
| 求第 k 小 | 支持（权值 BIT + 倍增） | 支持 |

**总结**：
- 如果问题只是**单点更新 + 区间求和**，树状数组是首选。
- 如果需要**区间更新 + 区间求和（加减操作）**，用二阶树状数组仍是好选择。
- 如果需要 **max/min/gcd** 等非可逆运算，或需要复杂的懒标记逻辑，用线段树。

## 八、模板代码

### C++ 完整模板

```cpp
#include <bits/stdc++.h>
using namespace std;

class BIT {
    int n;
    vector<int> tree;

public:
    BIT(int n) : n(n), tree(n + 1) {}

    void add(int idx, int delta) {
        for (int i = idx; i <= n; i += i & -i)
            tree[i] += delta;
    }

    int sum(int idx) {
        int res = 0;
        for (int i = idx; i > 0; i -= i & -i)
            res += tree[i];
        return res;
    }

    int range_sum(int l, int r) {
        return sum(r) - sum(l - 1);
    }

    // 求最小的 idx 使得 sum(idx) >= k（1-indexed）
    int kth(int k) {
        int pos = 0;
        for (int d = 1 << (__lg(n) + 1); d > 0; d >>= 1) {
            int nxt = pos + d;
            if (nxt <= n && tree[nxt] < k) {
                k -= tree[nxt];
                pos = nxt;
            }
        }
        return pos + 1;
    }
};
```

### Python 模板

```python
class BIT:
    def __init__(self, n: int):
        self.n = n
        self.tree = [0] * (n + 1)

    def add(self, idx: int, delta: int) -> None:
        while idx <= self.n:
            self.tree[idx] += delta
            idx += idx & -idx

    def sum(self, idx: int) -> int:
        res = 0
        while idx > 0:
            res += self.tree[idx]
            idx -= idx & -idx
        return res

    def range_sum(self, l: int, r: int) -> int:
        return self.sum(r) - self.sum(l - 1)
```

## 九、易错点总结

1. **1-indexed 约定**：树状数组下标从 1 开始，0 是哨兵（`sum(0) = 0`）。如果原数组从 0 开始，需要转换。
2. **离散化**：权值树状数组的值域过大时需要离散化。
3. **数据类型溢出**：求和结果可能超出 int，使用 `long long`。
4. **区间端点处理**：`range_add(l, r, v)` 时注意 `r+1 <= n` 的判断。
5. **查询空区间**：`sum(0)` 必须返回 0，否则死循环。
6. **更新顺序**：在求第 k 小的倍增法中，`tree[pos + d]` 存的是 `(pos+1 .. pos+d]` 的和，注意严格小于才跳。
7. **二维 BIT 初始化**：二维 BIT 需要 O(n×m) 空间，大数组时要注意内存。

## 十、扩展思考

### 10.1 树状数组为何不能求区间 max？

因为 max 运算**不可逆**。前缀和是可逆的（`a[l..r] = sum(r) - sum(l-1)`），但 max 不行。线段树之所以能求区间 max，是因为它将区间分解为若干不相交子区间，而非前缀相减。

### 10.2 树状数组与二进制的关系

树状数组的每个索引 `i` 的 `lowbit` 决定了它负责的区间长度。本质上，树状数组是一个**基于二进制拆分的分治结构**——它将前 `i` 个元素按 `i` 的二进制表示分解为若干段，每段长度都是 2 的幂。

### 10.3 树状数组能替代线段树的场合

当维护的信息满足：
1. **结合律**（associative）：`(a∘b)∘c = a∘(b∘c)`
2. **存在逆元**（invertible）：对每个 `a` 存在 `a⁻¹` 使得 `a∘a⁻¹ = 0`（单位元）

此时可以用树状数组。例如：加法、异或（XOR）、模加法等。

若不满足逆元（如 max, min, gcd），则只能用线段树。

### 10.4 Fenwick Tree 的变种

- **Range Update Point Query (RUPQ)**：差分 BIT
- **Range Update Range Query (RURQ)**：双 BIT
- **Fenwick Tree on Fenwick Tree**：二维 BIT
- **Fenwick Tree + 离散化**：处理动态数据
- **持久化 Fenwick Tree**：可回滚的历史版本
- **二叉索引树（Binary Indexed Tree of Min/Max）**：特殊构造可求前缀最值（不可逆但可前缀）

## 十一、总结

树状数组是一种极其**精巧、轻量、高效**的数据结构。它用最简洁的代码实现了对数级别的更新和查询，在算法竞赛和面试中都是高频考点。

**核心要点只需三行：**

```
lowbit(x) = x & -x
add(pos, delta):  while (pos <= n) tree[pos] += delta, pos += lowbit(pos)
sum(pos):         while (pos > 0)  res += tree[pos], pos -= lowbit(pos)
```

掌握树状数组，不仅是学会了一个数据结构，更是理解了**二进制拆分**和**分治思想**的精妙应用。

> **"Less code, more thinking."** —— Fenwick Tree 的最佳注脚。
